# EduBot — Adaptive AI Learning Companion

AI-powered personalized learning platform built using IBM Granite 4 on Watsonx.ai.

EduBot adapts explanations based on:
- student level
- learning style
- learning history
- contextual RAG retrieval

Unlike traditional educational chatbots, EduBot behaves like an intelligent personal tutor with adaptive teaching and interactive learning support.

## Key Features
- Personalized AI explanations
- RAG-based contextual learning
- AI doubt solving
- MCQ quiz generation
- Progress tracking
- Voice input support
- PDF upload & export
- Revision notes & key points
- Learning history

## Tech Stack
- IBM Granite 4
- IBM Watsonx.ai
- Streamlit
- LangChain
- ChromaDB
- HuggingFace Embeddings

## Why EduBot is Different
Most AI tutors give the same response to every learner.

EduBot dynamically adapts explanations for:
- Beginners
- School students
- Diploma students
- BTech students
- Advanced learners

and supports multiple learning styles like:
- Visual learning
- Step-by-step learning
- Exam-focused learning
- Quick revision learning
